# Cloud Foundations Notes
Cloud computing is the on-demand delivery of computing resources such as servers, storage, databases, networking, and software over the internet. Instead of purchasing and maintaining physical hardware, users can access resources as needed and pay only for what they use.
---

## Basic Computing Concepts

### Hardware - physical components of a computer system:
* CPU
* Memory (RAM)
* Storage devices
* Network interfaces

### Software - programs and applications that run on hardware:
* Operating Systems
* Web Browsers
* Productivity Applications
---

## Development Team Roles

### Quality Assurance
### Project manager
### Analyst
### Security Engineer
### Software Developer
### Database Adminstrator

Cloud computing allows organizations to:
* Access resources on demand
* Scale resources quickly
* Reduce infrastructure costs
* Improve reliability and availability

### Cloud Service Models

#### Infrastructure as a Service (IaaS)
Provides virtual servers, storage, and networking.
Example: * Amazon EC2

#### Platform as a Service (PaaS)
Provides a platform for application development.
Example: * AWS Elastic Beanstalk

#### Software as a Service (SaaS)
Provides software applications over the internet.
Examples: * Gmail
* Microsoft 365

## What is AWS?
Amazon Web Services (AWS) is a cloud computing platform that provides over 200 services for computing, storage, networking, databases, security, and analytics.

Popular AWS services:
* Amazon EC2
* Amazon S3
* Amazon RDS
* Amazon VPC
* AWS Lambda
* IAM
---

## AWS Pricing Concepts
 Pay-As-You-Go
 Save When You Reserve
 Volume-Based Discounts
---

## AWS Global Infrastructure
### Regions
Geographic locations where AWS data centers are deployed.

### Availability Zones (AZs)
Separate data centers within a region.

### Edge Locations
Used by services such as CloudFront to deliver content closer to users.
---

## Amazon S3
Amazon S3 is an object storage service used to store and retrieve data.

Common use cases:
* Static website hosting
* Backups
* Media storage
* Disaster recovery

Benefits:
* Durable
* Scalable
* Highly available
---

## Amazon EC2
Amazon EC2 provides virtual servers in the cloud.

Key features:
* Multiple instance types
* Flexible scaling
* Secure access through key pairs and security groups

Common use cases:
* Web servers
* Application servers
* Development environments
---

## Key Takeaways
* Cloud computing provides flexible, scalable, and cost-effective resources.
* AWS offers a wide range of cloud services.
* S3 is used for storage.
* EC2 is used for virtual servers.
* Security is a shared responsibility between AWS and the customer.
* AWS infrastructure is built around Regions and Availability Zones.
